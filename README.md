# Tab Colors and Rename

Right-click any tab to rename it or give it a color. Works on all tab types — markdown files, terminals, graphs, search, and anything else.

## Usage

### Rename
- **Right-click a tab** > **Rename tab** — set a custom display name
- **Right-click a file tab** > **Rename file** — actually renames the file on disk (updates all internal links)
- Also available via command palette: **Rename active tab**

### Colors
- **Right-click a tab** > **Tab color** — pick from 7 colors (Red, Orange, Yellow, Green, Blue, Purple, Pink)
- Select **None** to remove the color
- Colors show as a colored underline and tinted title text

Both custom names and colors persist across Obsidian restarts.

## Why?

Terminal tabs all say "Terminal." Multiple search tabs all say "Search." This plugin lets you tell them apart at a glance — name them, color them, or both.

## Installation

Copy the plugin folder to `.obsidian/plugins/tab-rename/` and enable it in Settings > Community Plugins.
